## 🚀 Featured Projects

**[EcoAgente](https://github.com/pedroveloso25/EcoAgente)** — Intelligent energy optimization agent that detects anomalies in household consumption, forecasts the next 24h using ML, and generates savings alerts via Gemini AI. `Python` `Streamlit` `Gemini API`

**[VozPro — Biu](https://github.com/lucasrabay/VozPro)** — WhatsApp AI assistant that helps low-literacy workers build a professional résumé through voice conversation — no forms, no typing. Transcribes audio, rewrites it professionally, and delivers a PDF with real job listings. `Python` `FastAPI` `Gemini API` `Docker`

**[European Soccer DB](https://github.com/pedroveloso25/european_soccer_db)** — Full-stack analytics app with CRUD operations over 25k+ matches across 11 European leagues (2008–2016). Includes standings, player evolution charts and head-to-head comparisons. `Python` `PostgreSQL` `Streamlit`

**[Fake News Detector](https://github.com/pedroveloso25/fake-news-detector-)** — NLP classifier that distinguishes real from fake news articles using text preprocessing and machine learning. `Python` `Scikit-learn` `NLP`
